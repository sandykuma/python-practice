### Part 1: Multiple Choice Questions
*Choose the correct option for each question.*

**1. What will be the output of the expression `7 / 2` in Python?**
- A) `3`
- B) `3.5`
- C) `3.0`
- D) `4`

**2. Which operator is used for "Floor Division" (dividing and rounding down to the nearest whole number)?**
- A) `/`
- B) `%`
- C) `//`
- D) `**`

**3. If you run the code `"Python" * 3`, what is the result?**
- A) An error (TypeError)
- B) `"Python3"`
- C) `"Python Python Python"`
- D) `"PythonPythonPython"`

**4. Which of the following is an INVALID variable name in Python?**
- A) `total_price`
- B) `user_name1`
- C) `2nd_value`
- D) `_temp_variable`

**5. What happens if you try to execute `"Age: " + 25`?**
- A) It prints `"Age: 25"`
- B) It prints `"Age: 252525..."`
- C) It results in a `TypeError`
- D) It results in a `SyntaxError`

#### Part 1: MCQs
1. **B) 3.5** — In Python, the `/` operator always performs "true division" and returns a float.
2. **C) //** — The double slash `//` is the floor division operator; it removes the decimal part.
3. **D) "PythonPythonPython"** — This is called **string replication**. It repeats the string the specified number of times without adding spaces.
4. **C) 2nd_value** — Variable names cannot start with a number. They must start with a letter or an underscore.
5. **C) TypeError** — You cannot concatenate a string (`str`) and an integer (`int`) directly. You must convert the integer using `str(25)` first.
